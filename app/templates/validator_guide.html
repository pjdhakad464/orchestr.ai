Data Ops Validator Guide

Use this page to set up, run, and share the workbook validator for Excel, CSV, and Google Sheets.

01

What This Engine Does

Upload an Excel workbook or CSV, or connect a Google Sheet, validate the data, highlight failed cells in red, add comments with recommendations, and download the verified file with a Validation Summary sheet.

02

Open The Validator

Use /excel-validator on the same machine, or the LAN URL if the app is running in network mode.

03

Install And Run

Install dependencies with python -m pip install -e .[dev].

Run locally with python -m uvicorn app.main:app --host 127.0.0.1 --port 8000.

Run for other systems with --host 0.0.0.0.

04

Main Workflow

Choose an Excel file, CSV file, or Google Sheet URL, keep or edit the rules JSON, run validation, review red-highlighted cells and comments, then download the validated workbook.

05

Current Rule Coverage

The engine checks title metadata, DAR rules, talent formatting, movie release date and genre recommendations, Facebook URL patterns, and Wikidata and IMDb verification.

06

Network URL

If the app is started in LAN mode, other systems on the same network can open the validator using your machine IP, for example http://172.16.16.46:8000/excel-validator.

07

Troubleshooting

If the app does not open, confirm the server is running on port 8000. If another system cannot reach it, check that the app is running with --host 0.0.0.0 and that Windows Firewall is not blocking the connection.
